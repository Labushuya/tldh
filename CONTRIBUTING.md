# Contributing

`tl;dh` is privacy-first. Contributions must preserve the core promise: voice-note processing stays local and session-only.

## Required for changes

- Update tests for behavior changes.
- Update README/docs for user-visible changes.
- Update CHANGELOG for release-relevant changes.
- Call out privacy impact in PRs.

## Not acceptable without explicit design discussion

- Cloud speech-to-text
- Cloud summarization
- Analytics
- Telemetry
- Persistent transcript database
- Silent/background update checks
