## v0.3.0 goal

Replace the fake summarizer's transcript placeholder with a real local transcription spike for short WhatsApp OGG_OPUS voice notes.

## Acceptance criteria

- Decode accepted OGG_OPUS input to a local PCM format suitable for STT.
- Keep the existing 50 MB / 3 min / 10 min / 15 min guardrails intact.
- Start with one audio at a time, no parallel transcription.
- Use visible progress and a cancel path.
- Do not send audio, transcript or metadata to any cloud endpoint.
- Keep the app usable offline except for optional manual update checks.
- Produce a raw transcript card before introducing real TL;DR summarization.

## Non-goals for v0.3.0

- No perfect summarization yet.
- No multi-audio session reasoning yet.
- No background transcription.
- No model zoo or advanced model picker.
