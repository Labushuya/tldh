# tl;dh Brand

`tl;dh` uses the `Long story, short.` direction: long voice-note energy compresses into a short, usable signal.

## Current assets

- `banner.png` — GitHub README banner / repo hero
- `logo.png` — source app-logo reference
- `icon-concept.png` — compact app-icon reference

## Color direction

The primary accent is now oriented around `#a50b5e`, with darker berry/magenta surfaces and small hot-pink glow accents. The UI should stay premium, spacious and mobile-first rather than over-saturated.

## UI guidance

- Use the logo as the in-app brand reference.
- Keep dense cards away from the bottom display edge.
- Treat long waveforms as noisy input and short waveform marks as summary/output.
- Avoid cloud symbolism; tl;dh is local-first.
