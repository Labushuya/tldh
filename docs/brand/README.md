# tl;dh Brand

## Name

`tl;dh` — **too long; didn't hear**

## Visual direction

- dark premium background
- pulsating purple gradient lines
- waveform/cut-to-point language
- local-first, no-cloud feel
- sharp, readable, not overdesigned

## Assets

- `banner.svg` — primary GitHub README banner
- `banner.png` — raster export for previews/social usage
- `icon-concept.svg` — first app/repo icon concept
- `palette.md` — color palette
- `design-tokens.md` — reusable token definitions

## Usage

Do:

- keep the exact casing `tl;dh`
- keep high contrast
- use purple gradients as lines, not huge flat fills

Don't:

- write `TLDH` as the brand
- use cloud imagery as the primary metaphor
- overuse neon/glow effects
